# nem2-helper-web

NEM2 Useful Tools.

Node.js Express Application.

## Usage

```
git clone https://github.com/planethouki/nem2-helper-web
cd nem2-helper-web
npm i
npm start
```

then, open browser and access to http://localhost:3000.
